KDataPy Bandpass Filter Database Doc¶

This is still in development (sorta of... manpower issues has slowed it down.) The idea is that this will help a user to upload a bandpass filter document to the database for use in automated analysis routines that are planned to be implemented. Contact adam.cox@kit.edu if you want to work on this.

KDataPy.bandpass.uploadIIRFilter(binwidth, order, docid, freqs, password, filtertype='butter', bandtype='highpass', replace=False)[source]¶

the filtertype and bandtype variables should match the expected values in scipy.signal.iirfilter binwidth is in nanoseconds freqs is either a scalar or a list of size two.
