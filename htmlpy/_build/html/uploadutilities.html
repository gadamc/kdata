KDataPy Upload Utility Functions¶

KDataPy.uploadutilities.formatvalue(value)[source]¶

This function reformats ‘value’ in an expected way. It tests if value is a string, and tries to convert it into an integer or float, returning the integer of float value if possible. If ‘value’ is a string, it strips off any whitespace or double quotation marks. This function is used extensively to format values found in the Samba header or in various Excel spreadsheets that are uploaded to our databases.

KDataPy.uploadutilities.splitargs(argv)[source]¶

A simple arg parser. This is useful because argparse is not installed on some Python systems with version < 2.7. This supports two kinds of arguments. 1. a single value 2. a key=value pair.

For example, this works “option1 option2 keyword1=value1 keyword2=value2 “

But this function doesn’t support “switches” that have values like “-k 5”. you must use the “=” sign and NO spaces.

It returns a tuple of ( [args], {kwargs} ). In the example above, for example it would return

( [‘option1’, ‘option2’], {‘keyword1’:’value1’, ‘keyword2’:’value2’})

disclaimer: It is highly recommended that you test this function before completely trusting what it does!! If you want a real tool for parsing arguments, you should use/install argparse.
